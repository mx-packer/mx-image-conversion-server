# MX Image Conversion Server
## Introduction
MX Image Conversion Server(MX ICS) is a server application which provides image conversion APIs to MX Packer components.

## Structure
```
MX Image Conversion Server
┣━ convert
   ┣━ avif
   ┗━ png
```
