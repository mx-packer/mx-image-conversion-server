Introduction

The MX Image Conversion Server(MX ICS) provides image conversion APIs to MX Packer components.

Settings API

To edit settings, you need to make a GET call to the following url :
http://localhost:@port/settings/(sector)


QUERY PARAMETERS(sector: avif)

Field Type Description
q Integer(0~100) (default: 60) A Quality factor.
effort Integer(1~10) (default: 1) A integer to set the CPU effort.
use_lossless Boolean (default: false) A boolean to use lossless compression.
use_subsampling Boolean (default: true) A boolean to use chroma subsampling.

QUERY PARAMETERS(sector: png)

Field Type Description
q Integer(0~100) (default: 80) A Quality factor.
effort Integer(1~10) (default: 4) A integer to set the CPU effort.
compression_level Integer(0~9) (default: 6) A integer to set compression level.
use_interlace Boolean (default: false) A boolean to use interlace mode.

RESULT

A processing result

Field Type Description
code String A status code.
message String A error details message.

Conversion API

To convert images, you need to make a GET call to the following url :
http://localhost:@port/convert


QUERY PARAMETERS

Field Type Description
format String A format of an image to be returned: avif, png.
filepath String The path to a file you want to convert.

RESULT

A converted image or a processing result

Field Type Description
code String A status code.
message String A error details message.

Status Codes

The MX ICS API uses the following status codes:

Settings API

Status Code Meaning
S000 Processing succeeded.
S001 Invalid value. This error appears when the query parameter type does not match or the value exceeds the limit.

Conversion API

Status Code Meaning
C000(RESERVED) Processing succeeded.
C001 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
C002 Missing or invalid filepath. This error appears when the file does not exist or the path is invalid.
C003 Invalid value. This error appears when the query parameter type does not match or the value exceeds the limit.
C004 Image conversion error. This error appears when the image conversion fails.
