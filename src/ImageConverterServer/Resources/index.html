MX Image Converter Server

User Manual

Request to '/convert' using GET method. You can use the following factors.

  • *type : png / avif
  • *path : urlencoded file path
  • cachedn : cache duration(min, default 1)
  • q : 0~100 => Q(Quality) factor
  • effort : 0~9 => CPU effort
  • lossless : 0 => use lossless compression(only avif)
  • useSubsampling : 0 => use chroma subsampling(only avif)
  • compression : 0~9 => compression level(only png)
  • interlace : 0 => use interlace(only png)

※ The '*' sign is attached to the required factors.
